PyVista in Action
=================

Show how PyVista is already being used within several projects and can be used for all things visualization. (15 min for talk)

.. tip::

    This section of the tutorial was adopted from `External Examples <https://docs.pyvista.org/getting-started/external_examples.html>`_
    chapter of the PyVista documentation.

Here are a list of longer, more technical examples of what PyVista can do!
